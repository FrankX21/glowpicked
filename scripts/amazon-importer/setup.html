🔖 Étape 1 — Installer le bookmarklet

A

Affiche la barre de favoris dans Chrome

⌘ + Shift + B pour l'afficher/masquer

B

Glisse ce bouton dans ta barre de favoris

✨ GlowPicked — Import

↑ Glisse ce bouton vers ta barre de favoris Chrome

C

Utilisation

Va sur n'importe quelle page produit Amazon, clique sur le bookmarklet. Un fichier glow-import-ASIN.json sera téléchargé automatiquement.

⚙️ Étape 2 — Traiter l'import avec Claude

1

Vérifie que tu as la variable d'environnement

echo $ANTHROPIC_API_KEY
2

Lance le script depuis le dossier du projet

# Depuis ~/Desktop/glowpicked/ node scripts/amazon-importer/process-import.mjs \ ~/Downloads/glow-import-ASIN.json # Avec catégorie et tier pré-remplis (optionnel) node scripts/amazon-importer/process-import.mjs \ ~/Downloads/glow-import-B00TTD9BRC.json \ face-moisturizers budget
3

Le script génère automatiquement

✅ site/data/analyses/ASIN.json — GlowScore + analyse IA

📋 L'entrée produit à coller dans src/pages/reviews/[category].astro

🚀 Étape 3 — Build et deploy

# Depuis ~/Desktop/glowpicked/site/ npm run build # Si le build est bon → push sur GitHub (auto-deploy Netlify) git add -A && git commit -m "feat: add [product name]" git push
